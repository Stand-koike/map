# Shimoda Resort Map

Mapbox GL JS と Google スプレッドシートを利用した、下田エリア向けの観光スポットマップです。  
カテゴリ別フィルター、ニュースティッカー、多言語切り替え（日本語/英語）、スポット詳細モーダル、クーポン表示、現在地表示などを備えたシングルページアプリケーションです。

---

## 主な機能

- **インタラクティブマップ表示**
  - Mapbox GL JS (`outdoors-v12` スタイル) を利用して地図を表示
  - 3D ビルディングレイヤーや空（sky layer）などのビジュアルエフェクト
  - PC では左側にスポットカードのサイドバー、スマホでは下部の横スクロールカードとして表示

- **スポットデータの外部管理（Google スプレッドシート連携）**
  - Google スプレッドシートから JSONP (`gviz/tq`) 形式でデータ取得
  - 緯度経度、アイコン絵文字、画像 URL、説明文、カテゴリ、ステータス、ニュース、詳細文、クーポン内容などを1行ごとに管理
  - 日本語/英語の多言語カラムに対応

- **カテゴリフィルター**
  - 左上の「Filter」ボタンからカテゴリ別の絞り込みが可能
  - 「すべて / All」チェックで一括 ON/OFF
  - 選択中カテゴリに応じて、地図上のマーカーとカードリストを同期して更新

- **ニュースティッカー**
  - スプレッドシート上で「ニュース」が設定されているスポットのみ抽出して表示
  - 画面上部のティッカーで定期的にスポットニュースをローテーション表示
  - ティッカーのクリックで対象スポットにフォーカス
  - 一覧ボタンからニュース一覧モーダルを開き、任意のニュースを選んで地図フォーカス

- **スポットカード & 詳細モーダル**
  - スポットごとにカード UI を生成
    - サムネイル画像（エラー時は絵文字プレースホルダ）
    - ステータスバッジ（混雑/満/通常など、色付きで表示）
    - ニュースがある場合はバッジ表示
  - カードのクリックまたはマーカーのクリックでスポットを選択し、地図を移動
  - 矢印ボタンから詳細モーダルを開き、以下を表示
    - カテゴリ名
    - スポット名
    - 詳細説明（無ければ概要）
    - Google マップの経路案内リンク
    - クーポンボタン（クーポンが設定されている場合のみ表示）

- **クーポン表示**
  - 詳細モーダル内の「クーポンを使う / Use Coupon」ボタンからクーポン専用オーバーレイを表示
  - 店舗などで画面を提示する想定

- **現在地表示 & コントロール**
  - ブラウザの Geolocation API を利用してユーザーの現在地を追跡
  - 右下のボタン
    - 「GPS」ボタン: 現在地にフォーカス
    - 「マップ」ボタン: 表示中カテゴリのスポット全体が入るようにフィット

- **多言語対応（日本語 / 英語）**
  - 左上の言語ボタンで `JP` / `EN` をトグル
  - スプレッドシートの日本語/英語カラムを切り替えて表示
  - Mapbox の地物ラベルも `mapbox-gl-language` プラグインで言語切り替え

- **Google Analytics（任意）**
  - gtag を利用したイベントトラッキング（スポット選択、クーポン使用）
  - 計測用 ID (`G-XXXXXXXXXX`) は任意の GA4 プロパティに差し替え

## 前提環境

- **ブラウザ**
  - 最新版の Chrome / Edge / Safari / Firefox を想定
  - Geolocation API 使用のため、HTTPS 環境推奨（ローカルは例外）

- **外部サービス**
  - **Mapbox**
    - 有効な Mapbox アカウント
    - 有効なアクセストークン (`mapboxgl.accessToken`)
  - **Google スプレッドシート**
    - 公開（匿名閲覧可能または適切な公開設定）のスプレッドシート
    - `gviz/tq` エンドポイントへのアクセスが許可されていること
  - **（任意）Google Analytics**
    - GA4 の計測 ID

- **開発環境**
  - 必須のビルドステップはなく、静的 HTML として動作
  - 任意で以下のような HTTP サーバを利用可能
    - VSCode/Cursor 拡張の `Live Server`
    - `npx http-server` などの簡易サーバ

## セットアップ手順

1. **リポジトリを取得**
   ```bash
   git clone <このリポジトリのURL>
   cd <リポジトリ名>
   ```

2. **`index.html` の設定を編集**
   - `CONFIG.MAPBOX_TOKEN` に自分の Mapbox トークンを設定
   - `CONFIG.SHEET_ID` に自分の Google スプレッドシート ID を設定
   - 必要に応じて GA4 の計測 ID (`G-XXXXXXXXXX`) を差し替え

3. **スプレッドシートの構成を準備**
   - 以下のカラム構成をもつシートを用意（列番号はコード中の `COLS` と対応）
     - `NAME`（スポット名 / 日本語）
     - `LAT`（緯度）
     - `LNG`（経度）
     - `EMOJI`（地図上に表示するアイコン絵文字）
     - `URL`（画像 URL）
     - `DESC`（概要）
     - `CAT`（カテゴリ名）
     - `HIDDEN`（非表示フラグ）
     - `STATUS`（ステータス：混雑 / 満 / 通常 など）
     - `NEWS`（ニュース文 / ティッカー用）
     - `DETAIL`（詳細説明）
     - `COUPON`（クーポン文）
     - `NAME_EN` / `DESC_EN` / `CAT_EN` / `NEWS_EN` / `DETAIL_EN` / `COUPON_EN`（英語版各種）
   - 公開設定を行い、ブラウザから JSONP で参照可能な状態にする

4. **ローカルで閲覧**
   - ファイルをダブルクリックしてブラウザで開くか、
   - または簡易 HTTP サーバ経由で `index.html` にアクセス

## 使い方

- **言語切り替え**
  - 左上の `JP` / `EN` ボタンで UI 表示と言語を切り替え

- **カテゴリフィルター**
  - `Filter` ボタンでパネルを開き、カテゴリごとのチェックボックスを操作
  - 「すべて / All」で一括 ON/OFF

- **スポットの閲覧**
  - 地図上のマーカーをクリックするか、カードをクリックしてスポットを選択
  - スポット選択時に地図がその地点へ移動し、カードがハイライトされる
  - 矢印ボタンから詳細モーダルを開いて詳細情報とクーポン（あれば）を閲覧

- **ニュースの確認**
  - 画面上部のニュースティッカーをクリックして、ニュース対象スポットへ移動
  - 右側のリストアイコンからニュース一覧モーダルを開いて任意のニュースを選択

- **クーポンの利用**
  - 詳細モーダル内のクーポンボタンを押してクーポン画面を表示
  - 店舗などで画面を提示して利用

- **現在地表示**
  - ブラウザの位置情報アクセスを許可すると、地図上に現在地アイコンが表示
  - 右下の GPS ボタンで現在地へフォーカス

## デプロイ方法の例

静的サイトとしてホスティング可能です。以下のようなサービスを想定しています。

- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages
- 任意の静的ホスティング環境

`index.html` をルートに配置し、必要であればカスタムドメインを設定してください。

## 注意事項

- Mapbox・Google スプレッドシート・Google Analytics の利用規約・料金体系に注意してください。
- 公開するスプレッドシートには個人情報や機微情報を含めないでください。
- ブラウザの位置情報はユーザーの同意が必要であり、利用目的を明示することを推奨します。
